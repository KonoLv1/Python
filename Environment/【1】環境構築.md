# 環境構築

## Pythonを開発するには一般的に下記のツールが必要となる

* **Python**

Pythonのコードを実行する

* **テキストエディタ**

## 今回は下記のツールを使う

* **Atom**

1. Pythonをダウンロードする（2019年現時点で健在）

2. **Add Python 〇〇 to PATHをチェックしてインストール

3. コマンドプロンプトを起動する（Windows画面左下でcmdを入力、黒い画面が表示される）

4. **python -version**と入力する（バージョンが表示されれば無事インストールできている）

5. Atomをダウンロードする

6. インストール

7. Atomで〇〇.pyというファイルを作りコード入力

```python
print('Hello World')
print(1 + 2 + 3 + 4 + 5)

```

12. コマンドプロントで実行
